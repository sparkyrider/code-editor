# KnotCode v1.11.0 — Release Copy

## Release Headline

**KnotCode 1.11: Cross-Platform Release — macOS, Windows, Linux, and iOS-ready.**

## Short Description (one-liner)

A lightweight, AI-native code editor that runs on every major platform. 7.8 MB. Free forever.

## Long Description

KnotCode v1.11 is the cross-platform release: comprehensive UI/UX polish across every surface, production-grade desktop release automation for macOS, Windows, and Linux, and iOS hardening for TestFlight. Every interaction has been refined — from theme consistency to accessibility, from real chat history to cross-platform keyboard shortcuts. The release pipeline now generates SHA256 checksums, structured release notes, and conditional signing for all three desktop platforms.

## Key Stats

- 7.8 MB app size (vs ~450 MB for Electron-based editors)
- ~80 MB RAM idle (vs ~750 MB competitor average)
- <1s startup time (vs 3-8s for competitors)
- 24 built-in themes
- 100% local data — nothing sent to the cloud
- Free forever, open source (Apache 2.0)

## App Store Description (iOS — draft)

**KnotCode — AI Code Editor**

A lightweight, AI-native code editor for developers on the go. Connect to your own AI gateway, browse repositories, edit code, review diffs, and commit changes — all from your phone.

Features:

- AI agent chat with streaming responses
- Repository browsing and file editing
- Diff review with accept/reject per hunk
- GitHub integration: commit, push, branch switching
- 24 built-in themes
- No subscription required — free forever

KnotCode uses your own AI gateway through OpenClaw. Your keys, your models, your data stays on your device. No cloud dependency, no telemetry, no tracking.

Requires iOS 17.0 or later.
